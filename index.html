My Little Story

Saya memiliki hobi bermain Mobile Legend. Mobile Legend merupakan game online yang dimainkan secara tim. Dalam satu tim terdiri atas 5 orang dengan durasi main 15 menit sampai 20 menit untuk 1 ronde. saya bermain Mobile Legend sejak kelas 8 SMP tepatnya pada tahun 2018. Saya memiliki hero favorit yaitu dyrroth, dyrroth saya sudah pernah menenpati peringkat 1 di kecamatan kesamben dan menempati peringkat 19 di kabupaten blitar. Awal nya saya main Mobile Legend hanya untuk pelampiasan emosi tapi semakin lama malah menjadi emosi kalau tidak main Mobile Legend. Sudah bertahun-tahun saya memainkannya, itu membuat saya candu. Akhirnya pada satu waktu saya sangat tertarik mengenai bahasa pemrograman dan mulai mempelajarinya secara otodidak, dari situ tanpa disangka saya menjadi kurang tertarik dan merasa bosan dengan Mobile Legend. saat itu saya memulai belajar bahasa pemrograman dari HTML dan CSS kemudian sempat lanjut ke dasarnya javascript tapi ternyata disitu saya mulai menemukan kesulitan dan bingung mau bertanya ke siapa sampai akhirnya stuck dan memilih kembali fokus untuk push rank di Mobile Legend. Tetapi karena saya bercita-cita menjadi seorang front end developer jadi beberapa bulan kemudian saya mulai bangkit dan mempelajarinya secara perlahan tanpa meninggalkan hobi saya sampai saat ini dan selamanya.
